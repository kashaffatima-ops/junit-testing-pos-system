# POS System Testing and Code Coverage Analysis

## Overview

This repository contains a comprehensive software testing and code coverage analysis project conducted on a Point of Sale (POS) System. The project focuses on verifying system functionality through JUnit test cases and evaluating code quality using statement, decision, and condition coverage metrics generated through EclEmma.

The objective was to identify defects, validate expected system behavior, analyze code coverage, and recommend improvements to enhance software reliability and maintainability.

## Tools Used

* Java
* JUnit Testing Framework
* EclEmma Code Coverage Tool
* Eclipse IDE
* Object-Oriented Programming (OOP)

## Project Components

### Unit Testing

Test cases were developed and executed for the following classes:

* POS
* PointOfSale
* POSSystem
* POR
* POH
* Inventory
* Employee
* EmployeeManagement
* Item
* ReturnItem
* Management

### Coverage Analysis

The project includes analysis of:

* Statement Coverage
* Decision (Branch) Coverage
* Condition Coverage

Coverage reports were generated using EclEmma and evaluated to determine tested and untested code paths.

### Test Results

The repository contains:

* Test Case Summaries
* Pass/Fail Analysis
* JUnit Execution Results
* Coverage Reports
* Coverage Gap Analysis
* Recommendations for Improved Test Coverage
